# 🤖 AI Assistant Usage in Student Life

An **Exploratory Data Analysis (EDA)** project examining how students across different academic disciplines use AI assistants — what tasks they use them for, how often, and how satisfied they are.

---

## 📌 Objective

To analyze and visualize patterns in AI assistant adoption among students, including:
- Which disciplines use AI tools most
- What academic tasks AI is most commonly used for
- Satisfaction levels and frequency of usage across student groups

---

## 📂 Repository Structure

| File | Description |
|------|-------------|
| `sample.ipynb` | Jupyter notebook with full EDA — cleaning, analysis, and visualizations |
| `ai_assistant_usage_student_life.csv` | Raw dataset |
| `cleaned_ai_student_data.csv` | Cleaned and preprocessed dataset |
| `ai-assistant-usage-in-student-life.pbix` | Power BI dashboard for interactive visualization |
| `requirements.txt` | Python dependencies |

---

## 🔍 Key Analysis Areas

- **Usage frequency** — How often do students turn to AI tools?
- **Task distribution** — Essay writing, coding, research, summarization, etc.
- **Discipline-wise breakdown** — STEM vs. Arts vs. Social Sciences
- **Satisfaction analysis** — Are students happy with AI assistance?
- **Trends & correlations** — Relationships between usage patterns and outcomes

---

## 🛠️ Tech Stack

- **Python** — pandas, matplotlib, seaborn
- **Jupyter Notebook** — EDA and storytelling
- **Power BI** — Interactive dashboard (`.pbix`)

---

## 🚀 How to Run
```bash
git clone https://github.com/Aishwarya-J05/Ai-assistant-usage-in-student-life.git
cd Ai-assistant-usage-in-student-life
pip install -r requirements.txt
jupyter notebook sample.ipynb
```

For the Power BI dashboard, open `ai-assistant-usage-in-student-life.pbix` in **Microsoft Power BI Desktop**.

---

## 📊 Dataset

The dataset contains survey/usage data from students with fields such as student discipline, AI tool used, frequency of use, task type, and satisfaction rating.
